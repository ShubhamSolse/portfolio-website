Fake News Detection using Fuzzy Deep Hybrid Network

Comprehensive fake news detection system using Fuzzy Deep Hybrid Network (FDHN) following CRISP-DM methodology. Implemented 8 model variants comparing Gaussian, Triangular, Trapezoidal, Sigmoid, and Bell-shaped fuzzy membership functions. Integrated BERT embeddings with fuzzy logic for uncertainty handling. Trained on LIAR2 dataset with 23,000+ fact-checked political statements. Features ablation studies, statistical significance analysis, and comprehensive evaluation with accuracy, F1-score metrics.
